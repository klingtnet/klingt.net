## Installation

```sh

```

Install hugo static site generator:

```sh
go get -v github.com/spf13/hugo
```

Take a look at the [official installation instructions for hugo](https://gohugo.io/overview/installing/) to get more details.

## Build

```sh
make
```

## Deploy

```sh
make deploy
```
